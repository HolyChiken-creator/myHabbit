# Stage 7 — Achievement UI & Animations

- Added a game-style achievement notification that slides in from the upper-left.
- Every notification contains the achievement icon, title, rarity glow, and optional XP reward.
- Notifications remain visible for five seconds and leave to the right.
- Added a queue so multiple achievements never overlap.
- Added rarity styling for Common, Uncommon, Rare, Epic, Legendary, and Mythic.
- Added a lightweight generated chime, with richer sequences for Legendary and Mythic achievements.
- Tapping a notification opens the Achievements screen.
- Swiping right dismisses a notification immediately.
- Added reduced-motion support.
- Collection milestone achievements now use the same central unlock flow and notification system.
- Updated app and service-worker cache versions.
