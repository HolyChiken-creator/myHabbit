# myHabbit v6.0 — Stage 3

## Animated profile cosmetics

- Added 8 animated avatar/profile frames: Starlight, Sakura, Hearts, Neon, Fire, Frost, Rainbow, and Night Bats.
- Added 8 animated nickname effects: Golden Shimmer, Living Gradient, Neon, Wave, Fire, Frost, Hologram, and Bouncy Letters.
- Added separate equipment slots for a normal frame, animated frame, badge, nickname effect, and theme.
- Added automatic migration for existing local profiles and automatic catalog merge, so old saves receive the new cosmetics without resetting progress.
- Animated cosmetics are visible both in the personal profile and family member cards.
- Added profile customization selectors and a preview block.
- Only one animated frame and one nickname effect can be active at a time.
- Added reduced-motion support; animations become static when the operating system requests less motion.
- Updated the PWA cache/version marker.
