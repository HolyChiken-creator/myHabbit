# myHabbit Content Library

Готова JSON-бібліотека для інтеграції у Cloudflare Worker / PWA.

## Вміст

- 5 500 унікальних квестів
- 700 щоденних повторюваних справ
- 350 щотижневих справ
- 450 довгих челенджів
- 1 200 досягнень
- 300 предметів магазину

Разом: 8 500 елементів.

## Інтеграція

1. Скопіюйте папку `content` або весь вміст архіву до проєкту.
2. Завантажуйте потрібні JSON-файли як статичний каталог.
3. У стані сім’ї зберігайте лише ID активних квестів, прогрес та історію.
4. Не дублюйте повні визначення квестів у Durable Object.
5. Нагороди перевіряйте на сервері за ID квесту з каталогу.
6. Для профілю зберігайте максимум три `featuredAchievementIds`.

## Повтори

- `quests/` — не повторюються автоматично.
- `daily/` — повторюються щодня.
- `weekly/` — повторюються щотижня.
- `challenges/` — мають власну тривалість.

## Важливо

Це велика стартова бібліотека, згенерована за шаблонами. Перед релізом бажано вручну переглянути баланс нагород і найважливіші категорії.
